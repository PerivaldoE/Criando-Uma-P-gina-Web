Surgimento da Internet

A Internet surgiu a partir de uma necessidade de comunicação segura e eficiente entre pesquisadores e militares durante a Guerra Fria. O embrião da Internet foi a ARPANET (Advanced Research Projects Agency Network), um projeto iniciado pelo Departamento de Defesa dos Estados Unidos em 1969. A ARPANET permitia a comunicação entre computadores de diferentes universidades e instituições de pesquisa, criando uma rede de troca de informações científicas e tecnológicas.

Surgimento do HTML

O HTML, ou Hypertext Markup Language (Linguagem de Marcação de Hipertexto), é a linguagem fundamental que permite a criação e formatação de documentos hipertexto na World Wide Web. Seu surgimento está intrinsecamente ligado ao desenvolvimento da própria web.

A história do HTML remonta ao início da década de 1990, quando Tim Berners-Lee, um cientista do CERN (Organização Europeia para a Pesquisa Nuclear), criou o sistema da World Wide Web. Berners-Lee desenvolveu o HTML como uma forma de compartilhar informações de pesquisa de forma mais eficiente entre os colegas do CERN.

O primeiro documento HTML foi publicado por Tim Berners-Lee em 1991. Esse documento descrevia a estrutura básica de uma página web usando tags simples de marcação. Incluía recursos como links para outros documentos e imagens, permitindo que os usuários navegassem facilmente entre diferentes partes de um documento.

Visual Studio Code

O Visual Studio Code (VS Code) é um editor de código-fonte desenvolvido pela Microsoft. Ele é gratuito, open-source e disponível para Windows, macOS e Linux. É uma das ferramentas onde faço todo minha estrutura de codigo para web.

Inspetor de Ferramentas

O Inspetor de Elementos é uma funcionalidade incluída nas ferramentas de desenvolvimento dos navegadores (como Google Chrome, Mozilla Firefox, Microsoft Edge e Safari). Ele permite visualizar a estrutura do DOM (Document Object Model) e o CSS aplicado aos elementos de uma página web, bem como realizar edições temporárias no código para testar mudanças instantaneamente.

Estrutura basica do HTML

  1. !DOCTYPE html:
    • Declara o tipo de documento e a versão do HTML que está sendo usada. Em HTML5, apenas !DOCTYPE html é necessário.
  2. html lang="pt-br":
    • A tag html é a raiz do documento HTML. O atributo lang="pt-br" especifica o idioma do conteúdo (português do Brasil).
  3. head:
    • A tag head contém metadados sobre o documento, como título, codificação de caracteres, e links para estilos e scripts.
  4. meta charset="UTF-8":
    • Define a codificação de caracteres como UTF-8, que é uma codificação de texto amplamente usada.
  5. eta name="viewport" content="width=device-width, initial-scale=1.0":
    • Configura a visualização para que a página seja exibida corretamente em dispositivos móveis, ajustando a largura da página à largura da tela do dispositivo.
  6. title> Minha Página Web title>:
    • Define o título da página que aparece na aba do navegador.
  7. body:
    • A tag body contém todo o conteúdo visível da página web, como texto, imagens, links, etc.
  8. header:
    • A tag header geralmente contém cabeçalhos de navegação ou introduções.

Tags

As tags HTML são os elementos fundamentais de qualquer página web. Elas formam a estrutura do conteúdo e instruem os navegadores sobre como apresentar e organizar os dados. Vamos explorar algumas das tags HTML mais comuns, agrupadas por categorias.

Atributos das Tags

Os atributos das tags HTML são usados para fornecer informações adicionais sobre elementos HTML. Eles são incluídos dentro da tag de abertura e geralmente consistem em um nome e um valor. Aqui está uma visão geral de alguns dos atributos mais comuns e como eles são utilizados:

Atributos Comuns

  1. id
    • Usado para identificar de forma única um elemento.
  2. class
    • Usado para classificar elementos em grupos, permitindo a aplicação de estilos CSS e manipulações JavaScript.
  3. style
    • Usado para aplicar estilos CSS diretamente ao elemento.
  4. title
    • Fornece um texto adicional sobre o elemento, geralmente mostrado como uma dica (tooltip) ao passar o mouse.
  5. href
    • Especifica o URL para o qual um link aponta.
    • Usado em a>.
  6. src
    • Especifica o caminho para um arquivo de imagem, vídeo, ou script.
    • Usado em img, video, audio, e script.
  7. alt
    • Fornece um texto alternativo para uma imagem.
  8. type
    • Especifica o tipo de elemento.
    • Usado em button, input, script, e style.
  9. name
    • Define o nome de um elemento, usado para identificar elementos dentro de um formulário.
    • Usado em input, form, select, textarea.
  10. value
    • Define o valor inicial de um elemento.
    • Usado em input, option, button.
